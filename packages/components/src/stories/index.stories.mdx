## Usage

<br />

### Installation

Before installing the component library, you'll need to
[install React](https://reactjs.org/docs/getting-started.html). Afterwards, run
the following command from your project root:

```zsh
yarn add @modelbcompany/woodmont-ui
```

### Transpiling Modules

Modules are written using
[ESNext syntax](https://www.javascripttutorial.net/es-next/) and need to be
transpiled before using them in a project. Add the following to your
[Babel configuration](https://babeljs.io/docs/en/configuration):

```json
{
  "exclude": "/node_modules/(?![@modelbcompany/woodmont-ui])/"
}
```

**Example `babel.config.json`**:

```json
{
  "env": {
    "test": {
      "plugins": [
        [
          "@babel/plugin-transform-runtime",
          {
            "regenerator": true
          }
        ]
      ]
    }
  },
  "exclude": "/node_modules/(?![@modelbcompany/woodmont-ui])/",
  "ignore": ["./node_modules/*"],
  "plugins": [
    "@babel/plugin-proposal-export-default-from",
    "@babel/plugin-proposal-export-namespace-from",
    "@babel/plugin-proposal-nullish-coalescing-operator",
    "@babel/plugin-proposal-optional-chaining",
    "@babel/plugin-proposal-throw-expressions",
    [
      "@babel/plugin-transform-typescript",
      {
        "allowNamespaces": true
      }
    ]
  ],
  "presets": [
    [
      "@babel/preset-env",
      {
        "modules": false
      }
    ],
    "@babel/preset-react"
  ]
}
```
